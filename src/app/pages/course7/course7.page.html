Python Course

Python is a cross-platform scripting language that runs on various platforms, including Windows, macOS, and Linux. Additionally, it has been adapted for Java and.NET virtual machines. It is both free and open-source. Python is a widely-used, general-purpose programming language that may be applied to several tasks (Opensource.com, n.d.). It offers high-level data structures, dynamic typing, dynamic binding, and other characteristics that make it ideal for scripting and "glue programming" connecting components. It can also be expanded to make system calls on virtually all operating systems and execute C or C++ programs. Python is a universal programming language due to its widespread use and ability to run on practically every system architecture. It is utilized in many applications.

Even though Python is pre-installed on most modern Linux and Mac computers, the version may be outdated. Therefore, it is always advisable to install the most recent version.

 

About Python

Guido van Rossum created Python in the late 1980s has advanced as an open-source programming language by controlling public discourse through Python Enhancement Proposals (PEPs). In 2018, van Rossum resigned as the language's Benevolent Dictator For Life (BDFL), and following PEP 13, a steering council was established to serve as the language's leader (Donat, 2015).

The Python Software Foundation (PSF) is a 501(c)(3) non-profit organization that owns the programming language Python's intellectual property rights. This comprises Python 2.1 and beyond, the Python Package Index (PyPI), the CPython reference implementation, and the infrastructure required to maintain the language. In addition to providing scholarships for software craftsmanship, the PSF hosts many PyCon conferences annually.

 

Python is now on its third major release and is updated often.

Your first Python Program

Now that Python is operational, we can write our initial Python application.

Let's construct a program with the name Hello World. A "Hello, World!" is a straightforward program that displays "Hello, World!" on the screen. Due to its simplicity, this program is frequently used to introduce newcomers to a new programming language.

Enter the code below into any text editor or IDE and save the file as hello_world.py.

print("Hello, world!")


Run the file, then. You will receive the output shown below.

>>Hello, world!

 

Kudos! You have just written your first Python program.

Python Identifiers

A name assigned to entities such as classes, functions, variables, etc., is called an identifier. It aids in distinguishing one entity from another.

Standards for composing identifiers

  1. Identifiers may consist of a combination of lowercase (a to z) or capital (A to Z) characters, numerals (0 to 9), or underscore (_). Valid examples are myClass, var 1, and print this to the screen.
  2. No identifier can begin with a number. 1variable is invalid, although variable1 is a valid name.
  3. Keywords cannot be used as identifiers.

Things to Remember

Python is a case-sensitive language. This implies that the variable and Variable are not identical. Always give the identifiers meaningful names. While c = 10 is a legal variable name, writing count = 10 would make more sense, and it would be easier to determine what it represents after a lengthy absence.
